CSV to Excel Converter

CSV to Excel Converter is a simple web app for converting csv file(s) to a excel file(with xlsx format).

Use it with following steps:

  1. Click upload button in in csv file upload area, then select csv file from file selector pop up.
  2. Fill the sheet name field in csv file upload area, this name will be the excel sheet name.
  3. If you want to merge multi csv file to a single excel file, click Upload More button below the upload area
